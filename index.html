n98-magerun

Provides some handy tools for magento developers like a quick database info from local.xml or a database dump feature, cache clearing, admin user password reset. You can also easily install a complete shop and sample data with buildin installer. For coders there are also many features like a module kickstarter (with modman support). If something is missing you can extend the tool with own commands.
